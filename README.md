# Tic Tac Toe  
### ❌⭕ Play. Think. Win.

> A sleek and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

---

##  Features
Smooth 2-player gameplay  
Instant win & draw detection  
Restart anytime  
Clean, minimal UI  

---

##  Built With
- HTML  
- CSS  
- JavaScript  

---

##  How to Play
- Player **X** starts first  
- Take turns placing your mark  
- Get **3 in a row** to win  
- No winner? It’s a **draw**
